# LinkedIn Batch Scheduler

Automated pipeline for scheduling and formatting LinkedIn posts.

## Key Features
- **Automated Scheduling**: Streamline the process of batching LinkedIn content.
- **Web UI**: User-friendly interface for managing the scheduling queue.
- **Queue Previews**: Full previews of upcoming posts with persistent media archiving.
- **Smart Formatting**: Automated formatting for LinkedIn's specific requirements.

## Setup and Usage
- Refer to `python_anywhere_guide.md` for deployment details.
- Run `run_pipeline.ps1` to start the processing queue locally.
